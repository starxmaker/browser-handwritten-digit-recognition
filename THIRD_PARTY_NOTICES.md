## ONNX Model Zoo MNIST model (`mnist-8.onnx`)

This library uses material derived from the ONNX Model Zoo MNIST handwritten digit recognition model.

- Source: https://github.com/onnx/models/tree/main/validated/vision/classification/mnist
- Model file: `mnist-8.onnx`
- Upstream license: MIT

The upstream model page identifies this model as MIT-licensed. No specific copyright holder is identified on the upstream model page.

`mnist-8.onnx` is processed during this library's build process to extract model weights and generate JavaScript source files used by the handwriting digit recognition algorithm.
